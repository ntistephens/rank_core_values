Privacy Policy

This tool is a simple application:

  • No user PII data is collected or stored.
  • All user data entry is stored only on the user's local computer, so no user data entry is collected or stored by the server.

Information Collection: None. The static website is stored on S3. Logs are not even enabled, so even IP addresses for requests are not stored.

Use of Information: N/A. No user information is collected.

Information Sharing: N/A. There is no data collected, so there is no data to share.

Data Protection: Website traffic is encrypted with HTTPS.

User Rights: Users are welcome to reach out to the website creator (contact information below).

Cookies and Tracking Technologies: This website does not inherently use cookies or tracking. This website does use open graph protocol (OGP) to increase user experience when sharing the link to this website on social media platforms. For example, when a user pastes the url to this website on Facebook: A small image representing the website will display on the post.

Changes to the Privacy Policy: Since this website collects no user information, there is no way to contact users on privacy policy changes. It is the responsibility of the user to check for updates to this privacy policy whenever the user accesses this website.

Contact Information: Feel free to reach out to the creator of the website on linkedin: https://www.linkedin.com/in/neilsthorne
